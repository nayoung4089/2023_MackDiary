> Link: https://nayoung4089.github.io/2023_MackDiary

## 🕰 개발기간
~ 2023.02.03 (약 5일)

## ⚙ 개발 환경
```React JS``` 
- npm: ``` react-canvas-draw ```
- database: ``` firebase ```

## 📌 주요 기능
- 구글/ 깃허브 로그인 
- Canvas로 맥진 모양 그리기 (그림판 기능)
- 수정/ 삭제 기능
- 공유 기능 (Global Page)

## 🎯 향후 계획
- ``` react-native ```와 ``` expo ```를 활용한 웹앱 구현 (개발 가능)
